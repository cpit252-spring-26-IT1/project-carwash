# Project Name
CarWash

## Description
The Car Wash Subscription System is an Android application designed to modernize the traditional car wash experience. It addresses the limitations of manual, on-site payments and lack of digital records by introducing a prepaid package model. The system allows customers to pay once for multiple services, eliminates the need for on-site transactions, and provides a digital framework for recording and supervising wash activities.


## Features
- Prepaid Wash Packages: Customers can purchase specific wash types (inside only, outside only or full wash) tailored to their vehicle type (Sedan, SUV or Crossover) directly within the app.

- Digital Verification: Fast service confirmation at the station using order ID or QR codes.

- Role-Based Access Control: Distinct interfaces and functionalities for three user types:

    -Customer: To purchase and manage wash packages.

    -Cashier: To verify customer packages and record completed washes.

    -Owner: To manage staff accounts and monitor daily wash activity reports.
- Service Records: Automatic digital logging of each wash, including details such as customer name, staff member, package type, date, and time.

- Loyalty & Promotions: Support for membership management, rewards, coupons, and discount codes.
